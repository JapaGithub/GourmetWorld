Gourmet World

Este proyecto es parte del curso Codo a Codo. La aplicación permite a los usuarios explorar recetas gastronómicas y cócteles de manera sencilla. Aquí tienes una breve descripción de las funcionalidades principales:


Página de Inicio

Muestra una receta y un cóctel aleatorios al cargar la página para inspirarte con nuevas ideas culinarias y de bebidas.


Sección de Gastronomía

Explora diversas recetas a base de pollo. Cada receta incluye una imagen, descripción y un botón para ver más detalles.


Sección de Cócteles

Descubre cócteles basados en ginebra. Cada cóctel se presenta con una imagen, descripción y un botón para obtener más información.


Modal de Detalles

Al hacer clic en "Ver más", se abre una ventana modal que muestra detalles adicionales de la receta o cóctel seleccionado, incluyendo ingredientes y pasos de preparación.


Tecnologías Utilizadas

HTML y CSS para la estructura y el estilo de la página.
JavaScript para la lógica de la aplicación y consumo de APIs.
Fetch API para obtener datos de TheMealDB y TheCocktailDB.
